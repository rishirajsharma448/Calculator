# 🧮 Calculator

A responsive and interactive calculator built using **HTML**, **CSS**, and **JavaScript**. This project performs basic arithmetic operations while providing a clean, modern, and user-friendly interface.

## 🚀 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear (AC) button
- ⌫ Delete (Backspace) functionality
- 📱 Responsive design
- ⚡ Fast and lightweight

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
